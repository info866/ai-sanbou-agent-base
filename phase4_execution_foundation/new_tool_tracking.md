## 記録フォーマット

### 新規ツール発見記録

```
## [ツール名]

**発見日**: [YYYY-MM-DD]
**発見経路**: [どこで見つけたか]
**参考リンク**: [URL]

### 判定1: 今回の作業に直接効くか？
- 結果: [YES / NO]
- 理由: [具体的な理由]

### 判定2: 既存フローより改善できるか？
- 結果: [YES / NO]
- 比較対象: [既存候補]
- 理由: [具体的な理由]

### 判定3: すぐ使えるか？（セットアップ時間 < 30分）
- 結果: [YES / NO]
- セットアップ時間: [分数]
- 内容: [何に時間がかかるのか]

### 判定4: セットアップ時間が許容か？（30分〜2時間）
- 結果: [YES / NO / 該当なし]
- 期待効果: [セットアップコストに見合う効果]
- 計画: [実施時期]

### 最終判定
- 採用判定: [即投入 / 試験導入 / 保留 / 不採用]
- 次アクション: [具体的な次ステップ]
```

### 試験導入実績記録

```
## [ツール名] - 試験導入実績

- [実施項目1]: [結果]
- [実施項目2]: [結果]

### 効果測定
- [定量指標]: [測定値]
- [定性評価]: [評価コメント]

### 見つかった課題
- [課題1]: [詳細]
- [課題2]: [詳細]

### 本格導入判定
- [ ] 本格導入（理由: [効果が十分]）
- [ ] 改善後に再試験（理由: [課題内容]）
- [ ] 中止（理由: [効果不足]）

### 記録日
[YYYY-MM-DD] [実施者名]
```
